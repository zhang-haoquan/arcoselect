## 开发图表配置弹窗界面

### 任务清单：

1. **安装 ECharts 依赖**
   - 运行 `npm install echarts` 添加图表库

2. **创建 ChartConfigModal 组件** (`src/components/ChartConfigModal.vue`)
   - 使用 Arco Design Modal 作为弹窗容器
   - 左右分栏布局（左侧图表预览 60%，右侧配置 40%）
   - 左侧：ECharts 图表渲染区域
   - 右侧：配置面板
     - Tabs 切换（数据与类型 / 自定义样式）
     - 图表类型选择（下拉框）
     - 数据范围设置（带筛选链接）
     - 横轴配置（维度、排序方式、排序规则）
     - 纵轴配置（统计指标）
   - 底部操作按钮（取消、确定）

3. **更新 App.vue**
   - 引入 ChartConfigModal 组件
   - 添加触发按钮和弹窗控制逻辑

4. **功能实现**
   - 配置项变更时图表实时更新
   - 响应式布局
   - 符合 Arco Design 设计规范
